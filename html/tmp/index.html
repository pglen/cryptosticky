To make the Personal Computer personal again.

   CryptoSticky is an easy to use computer memo application for the PC and Netbook. It is designed to empower you for the everyday computing tasks like quickly writing something down, doodling onto the memo pad or recording a voice memo. CryptoSticky also allows you to encrypt text memos with state of the art encryption. CryptoSticky's customizable look and feel, transparent user interface, seamless presence, and user friendly alarm makes this program an excellent choice. It stands out with a hierarchical tree of note categories, and an easy enable/disable show/hide feature for every note. CryptoSticky allows the re-use of previously created notes.

  No memo is lost. Deleted memos go through a trash / recyclers where they can be recovered from.CryptoSticky is also network enabled, and it can be used as a corporate messaging system. A memo created by one person can pop up on another person's computer. To find out more about CryptoSticky network messaging see the network memo page.
