  Creating a New CryptoSticky Memo

   There are multiple ways to create a new memo. CryptoSticky allows one to create a memo with a single mouse click, or by pressing a single hot key. Naturally, when creating a new memo, one has complete control over the memo's content and the memo's properties.

    1. Creating a memo from scratch:

    On the main panel, press the button 'Create new Memo' The following screen should show:

  In the 'Title' field: one may describe the memo with a short text. This text will be in the memo header and in the alarm dialog. For Example, the following titles are typical choices made by existing CryptoSticky users: 'Basketball practice' or 'Ebay Item:  Oscilloscope'. The title is well constructed if it is a clear and concise reminder of the content. There are no backslash ('\') characters allowed in the 'Title' field. Hotkey: Alt-I

  The 'Create Folder Memo' checkbox allows one to specify a container item. This enabled the item to be a 'leaf' or sub item, essentially constructing a hierarchical list. The parent of the leaf item is the current selection in the memo tree at the time of creation. Hotkey: Alt-F

  In the main text body one can enter arbitrary text. There is no limitation applied to the text, and the memos will be auto formatted and auto sized based upon content. The actual memo will have minimum and maximum screen size limits, in that case scroll bars will be shown.

When the 'Red Memo' check box is checked, the memo will have a red color, possibly communicating significance. Hotkey: Alt-R

When the 'Centered Text' check box is checked, the text of the memo is aligned in the middle of the memo. By default, the memo text is left aligned. Hotkey: Alt-C

When all desired entries are made, one can close this dialog by pressing the 'OK' button. The content of the memo is saved, the alarm queue is evaluated, and the alarm is placed on the alarm list. 

Pressing the 'Cancel' button will discard the memo contents. Hotkey: Alt-C.  To discard the current changes, press the 'Cancel' button. CryptoSticky will prompt you to confirm the discarding of the contents. Hotkey: ESC
   
2. Creating a memo from the clipboard:

    On the main panel, press the button 'Paste from Clipboard'. The same screen should show as above, with the contents of the clipboard already pasted into the main text body. One needs to enter a title for the memo before save. The memo is created at the current selection point in the memo hierarchy.

3. Creating avoice memo:

 To create a voice memo, just  press the  'Record'  button. While recording, the wave window gives a nice visual feedback of the sound and sound levels. To stop recording press the 'Record' button again or press the 'Play' button.
4. Doodle onto the memo:

  You can draw onto the memo's  surface with the mouse or pen. The drawing is with a single pen width and pen color, as a visual aid for the memo itself. No more drawing on napkins ... cryptomemo is here.

5. Shortcut Keys:

    Alt-O (synonym: OK) will close the memo editing dialog.
    Alt-I  (synonym: Title) Set focus to title window.
    Alt-R  (synonym: Red) Create Red memo.
    Alt-E  (synonym: cEntered) Create centered memo.
    Alt-T  (synonym: Top) will create an 'ontop' memo. Memo will appear on top of every other window.
    Alt-1  (synonym: 12 hour) set alarm dialog interaction 12 hour based   
    Alt-2  (synonym: 24 hour) set alarm dialog interaction 24 hour based
    Alt-W (synonym: Now) set alarm to Now + 1 Minute (next minute boundary)
    Alt-N (synonym: oN set alarm to ON. Press again to toggle
    Alt-S (synonym: Sound set alarm sound to ON. Press again to toggle
    Alt-B (synonym: Beep set alarm beep to ON. Press again to toggle

  There are a some  more shortcut keys on the main panel, one can view them by pressing the ALT key, and observing the underlined letters on the dialog.
