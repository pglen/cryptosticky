  CryptoSticky Future Development

    YellowMemo is entering the fast lane. We are working on the tablet version, where one can direct the whole operation from within the pen interface. We are also including a pen based gesture recognition and handwriting conversion. Some of the tablet features are already in YellowMemo, so it is easy to use it with a pen.

   The Pro version of YellowMemo is also in the works. It will allow monitoring  multiple memo folder sources for joining multiple team folders. It will have more extensive set of features like multimedia  fusion, to record voice memos and video memos. Porting it to the UNIX Platform and the OS X platform is planned.

  YellowMemo's server is in the making, where instead of monitoring the central location (or folder), the server will notify the client of the change in memo status. This will enable memo notes over the internet (TCP/IP) and instant memo delivery.

  YellowMemo will be equipped with an encryption facility where the notes (or part of the notes) will be encrypted with an industrial strength encryption. Only those who have the correct encryption key will be able to see the memo.

  Naturally, the feature list is endless. If you would like to request a feature that should be in the next release of YellowMemo please contuct us at the details found on our Contact page.
